# Learning Typescript 

## Install Requirement

### install typescript Globally

```npm i -g typescript```

### compiler command Line

```tsc <filename>```

### configure typescript compiler

```tsc --init```

if you have compiler configure running command

```tsc```

### different between javascript vs typescript

|javascript | Typescript|
|-----------|-----------|
|Number | any|
|string  |  unknown|
|boolean |never|
|null | enum|
|undefined | tuple|
|object|
